Python is a high-level, interpreted, object-oriented, and general-purpose programming language 
designed to be easy to read and write.

-------------------------------------------------

Python was created by Guido van Rossum in 1991.
Why he created Python ?
He wanted to create a language that:
1.Was easy to read and simple to learn
2.Reduced complexity of programming
3.Allowed developers to write code faster
4.Combined the power of languages like C with the simplicity of scripting languages
5.The name Python was inspired by the comedy show Monty Pythons Flying Circus, not the snake.

1. Compile Time

Compile time is the phase when the source code is translated into bytecode or machine code before the program starts executing.
During this phase, the compiler/interpreter checks for syntax errors.
Key Points:
Happens before execution
Converts code to bytecode or machine code
Detects syntax errors
No program output yet
Example (Python Syntax Error)
print("Hello"
Error:
SyntaxError: unexpected EOF while parsing
This error occurs before the program runs, so it is a compile-time error.

-------------------------------------------------

Run time is the phase when the program is actually executing after compilation.
During this phase, the program performs operations and produces output.
Key Points:
Happens during program execution
Program runs and produces output
Errors that occur during execution are runtime errors
Example (Runtime Error)
a = 10
b = 0
print(a / b)
Error:
ZeroDivisionError
This occurs while the program is running, so it is a runtime error.
